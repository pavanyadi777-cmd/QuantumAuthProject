QuantumAuthProject

Post-Quantum Cryptography Web Application built using Flask and Open Quantum Safe (liboqs-python) demonstrating secure communication using NIST-standardized quantum-resistant algorithms.

🚀 Features

* ML-KEM-512 (Kyber) Key Exchange
* ML-DSA-44 (Dilithium) Digital Signatures
* Secure Message Exchange Demo
* Message Encryption & Decryption
* Signature Verification
* Tampering Detection Demonstration
* Interactive Flask Web Interface

🛡 Security Algorithms

ML-KEM-512 (Kyber)

* Post-Quantum Key Encapsulation Mechanism
* Secure shared secret generation
* Quantum-resistant key exchange

ML-DSA-44 (Dilithium)

* Post-Quantum Digital Signatures
* Message authenticity verification
* Integrity protection

🔐 Secure Message Flow

1. User enters a message
2. ML-KEM generates a shared secret
3. Shared secret derives encryption key
4. Message encrypted using Fernet
5. ML-DSA signs encrypted message
6. Signature verification performed
7. Decrypted message displayed
